Seoripet

입양, 후원, 봉사 서비스를 통합한 유기동물 입양 플랫폼입니다.

프로젝트 소개

유기동물 입양 정보부터 후원과 봉사 참여까지 하나의 플랫폼에서 연결하여, 사용자가 여러 서비스를 오가지 않고 이용할 수 있도록 설계했습니다.

핵심 포인트

직관적인 입양 정보 탐색, 후원·봉사 서비스 연계, 반응형 UI 경험을 중심으로 구성했습니다.

사용 기술

HTML, CSS, JavaScript, Chatgpt, Supabase
