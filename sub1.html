PROJECT 01

Brand Website
Renewal

브랜드 아이덴티티를 반영한
웹사이트 리뉴얼 프로젝트입니다.

UI/UX Design Visual Design 2024

Overview

사용자 중심의 정보구조 재정비와 심플한 UI 개선을 통해 브랜드가 지향하는 감성과 방향성을 새로운 웹 경험으로 표현했습니다.

Goal

  • • 브랜드 아이덴티티 강화
  • • 사용자 경험 개선
  • • 콘텐츠와 인터랙션 명확화

Tools

  • F Figma
  • Ps Photoshop

01. Research

기존 사이트의 구조, 콘텐츠 흐름, 사용자 접근성을 분석했습니다.

02. Planning

핵심 콘텐츠를 재정리하고 메인 화면에서 보여줄 우선순위를 설정했습니다.

03. Publishing

HTML, CSS, JavaScript를 활용해 반응형 구조와 탭 인터랙션을 구현했습니다.

Visual Concept

네이비와 소라색 계열을 사용해 신뢰감 있고 차분한 브랜드 무드를 구성했습니다.

Layout

상단에는 프로젝트 핵심 이미지를 배치하고, 하단에는 탭을 통해 정보를 구분했습니다.

Result

브랜드 메시지가 명확하게 전달되도록 화면 흐름을 개선하고, 사용자 접근성을 높였습니다.

Point

핵심 기능과 디자인 포인트를 중심으로 포트폴리오에서 강조할 수 있도록 구성했습니다.
